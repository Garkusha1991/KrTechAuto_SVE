CheckDuplicate

1. CheckDuplicateAddEntranceExam
    Тест проверки на дубликат ВИ

2. CheckDuplicateSpecialities
    Тест проверки на дубликат создания специальности

3. CheckDuplicateStatement
    Тест проверки на дубликат заявления по Снилс + Серия Паспорта + Номер Паспорта


1. RegistrationClassic
Тест выполняет классическое заполнение всех полей, при регистрации нового заявления.

2. RegistrationComparisonFields
Тест выполняет проверку соответсвия заполненных полей при регистрации и полей в карточке заявления

3. RegistrationMandatoryFields
Тест выполняет проверку обязательности полей при регистрации заявления,
путём нахождения "Подсказок о необходимости заполнения полей", после нажатия кнопки "Далее".



Особенности:

Тесты (RegistrationClassic, RegistrationComparisonFields ) используются функции генерации рандомных:
СНИЛС, Серии документа, Номера документа.
Эти значения генерируются рандомно, что бы обходить проверку на дубликат заявления
